# IBM-EMPLOYEES-DATA-ANALYSIS-DASHBOARD

## 1. Overview
This project analyzes IBM employee data using an interactive dashboard to understand workforce demographics, income distribution, departmental strength, education background, and business travel patterns.

---

## 2. Key Insights
- Total Employees: 1.058K  
- Total Monthly Income: $7.01M  
- Gender-wise employee distribution  
- Department-wise employee count  
- Education fields across departments  
- Business travel trends and gender-based analysis  

---

## 3. Dashboard Highlights
- KPI cards for employees and income  
- Gender distribution chart  
- Business travel analysis  
- Department & education field comparison  

---

## 4. Tools Used
- Power BI   (Data Visualisation)

---

## 5. Conclusion
The IBM Employees Data Analysis Dashboard provides a comprehensive view of workforce composition and organizational trends. By analyzing employee count, income distribution, gender diversity, departmental strength, education background, and business travel patterns, the dashboard helps transform raw employee data into actionable insights. This project demonstrates the effective use of data visualization techniques to support data-driven decision-making in a corporate environment.
